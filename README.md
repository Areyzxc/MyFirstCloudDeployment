# My First Cloud Deployment

A simple personal website showcasing your name and profile picture.

## Features
- Clean, modern design with gradient background
- Responsive layout
- Profile picture display
- Beautiful typography

## Setup Instructions

1. Replace "Your Name" in `index.html` with your actual name
2. Replace "profile.jpg" with your own profile picture
3. Deploy to a web server or cloud platform

## Files Included
- `index.html` - Main website file with styling

## How to Use
1. Replace the image placeholder (`profile.jpg`) with your actual profile image
2. Update the name in the HTML file
3. Upload to your web hosting service

## Customization
You can customize:
- Colors: Modify the gradient colors in the CSS (look for `#667eea` and `#764ba2`)
- Name: Replace "Your Name" with your actual name
- Profile picture: Replace `profile.jpg` with your image file

## Deployment
This website can be deployed to:
- GitHub Pages
- AWS S3
- Azure Static Web Apps
- Any web hosting service
